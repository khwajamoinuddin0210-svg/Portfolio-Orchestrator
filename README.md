# AI Orchestrator Portfolio

A minimal, dark-mode-first portfolio for an AI enthusiast and AI orchestrator.

## Run Locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Edit Your Details

- Profile, GitHub, LinkedIn, and email: `src/config/profile.ts`
- Future projects: add items to the `projects` array in `src/config/profile.ts`
- AI news placeholders: update the `aiNews` array in `src/config/profile.ts`, or replace it later with an API route or external feed

## Build

```bash
npm run build
```
